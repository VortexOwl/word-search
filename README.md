# word-search
>Поиск слов русского языка с использованием фильтров поиска.

## Фильтры:
* Длина искомого слова;
* Список букв, которые отсутствуют в искомом слове;
* Список букв, которые присутствуют в искомом слове;
* Список букв, которые отсутствуют в определенных позициях искомого слова;
* Список букв, которые присутствуют в определенных позициях искомого слова.

## Запуск
### Активация скрипта из папки проекта
    python src/start.py
